# FrontMASS

> Thessaloniki FrontMASS event repository.
>
> [http://frontmass.org](http://frontmass.org)

## Install

Install required packages:

```shell
npm install
```

## Run On Local

Use grunt:

```shell
grunt server
```

## Build

Check out how the build site will look like:

```shell
grunt build
```

The output will be in the `dist/` folder.

## Deploy

Use grunt:

```shell
grunt deploy
```
