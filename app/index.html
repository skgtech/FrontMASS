About frontMASS

We're community

Building a web community in Thessaloniki is of utmost importance for us. We believe that frontend techologies are a wonderful thing to work with and we believe tha many people in Thessaloniki have the same feeling. We want to bring all of them together.

We're the web

We want to cover a variety of web related topics. From Javascript to the backend or tooling to purely CSS architecture, we hope that we can cover a very diverse interest spectrum.

We're proffessional

All the speakers are well respected professionals working in either some of the hottest greek startups or large international companies. Even though this is a non-commercial event, the people talking have real world experience in demanding, high-paced environments.

  • CSS architecture for large scale applications

    13:00 - 13:50

    Clean, maintainable, reusable, performant. These are non functional requirements that the codebase of any application is expected to meet. And of course CSS is no exception. In this talk, we will see how we can architect our CSS when building a web application in order to meet those requirements. We will focus on CSS structure, naming conventions, best practices and tooling which large companies use to build their large scale applications.
